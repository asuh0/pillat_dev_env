# Infra Templates

Этот каталог хранит централизованные шаблоны инфраструктуры, которые используются при генерации новых проектов.

## Содержимое

- `php/` — шаблоны `Dockerfile.php*` по версиям PHP.

## Правило

- Новые общие шаблоны добавляются сюда, а не в `projects/`.
- `projects/` должен содержать только каталоги реальных проектов.
