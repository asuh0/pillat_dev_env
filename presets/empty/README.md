# Пресет Empty

Минимальный пресет для быстрого создания "чистого" PHP-хоста без фреймворк-специфичных файлов.

## Что делает

- Применяет только базовую структуру проекта, которую создает `create-project.sh`.
- Не добавляет дополнительных шаблонов и файлов приложения.

## Пример использования

```bash
/scripts/create-project.sh my-empty.dev 8.2 mysql empty
```
