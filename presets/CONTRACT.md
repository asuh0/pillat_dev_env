# Preset Contract (MVP)

Для каждого пресета обязателен каталог `presets/<slug>/` и файл `preset.json`.

## Обязательные поля `preset.json`

- `name` - человекочитаемое имя пресета.
- `slug` - машинное имя пресета (должно совпадать с именем каталога).
- `version` - версия контракта/пресета.

## Опциональные поля `preset.json`

- `description` - описание пресета.
- `entrypoint.install` - путь до установочного скрипта внутри каталога пресета.
- `templates` - массив шаблонов, которые должны существовать внутри каталога.

## Правила валидации

1. Каталог пресета существует.
2. `preset.json` существует и является валидным JSON.
3. `name`, `slug`, `version` - непустые строки.
4. `slug` из метаданных совпадает с выбранным пресетом.
5. Если указан `entrypoint.install`, соответствующий файл существует.
6. Если указан массив `templates`, каждый файл из списка существует.

## Поддерживаемые пресеты MVP

- `empty`
- `bitrix`
